# HelloFlutter
[Flutter](https://flutter.dev/) basic learning

### Get started

1. 创建新的应用

```base
flutter create myapp
cd myapp
```

2. 运行应用程序

```base
# 检查 Android 设备是否在运行
flutter devices

# 运行应用程序
flutter run
```
